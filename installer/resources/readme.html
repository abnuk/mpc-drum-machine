Installation Locations

Component Location
Standalone App /Applications/Beatwerk.app
VST3 Plugin /Library/Audio/Plug-Ins/VST3/Beatwerk.vst3
Audio Unit /Library/Audio/Plug-Ins/Components/Beatwerk.component

System Requirements

macOS 11.0 (Big Sur) or later. Apple Silicon and Intel supported.

After Installation

After installing the plugins, you may need to rescan your plugins in your DAW for them to appear. The Audio Unit plugin will be registered automatically with macOS.

Uninstalling

To uninstall, run the included uninstall.sh script or manually remove the files from the locations listed above.
